# 📊 Analitik Raporlama ve Kullanıcı Deneyimi (UX)

Platform performansını izlemek ve kullanıcı deneyimini iyileştirmek için hazırlanan plandır.

## 1. İzlenecek Metrikler (KPIs)
* **Hemen Çıkma Oranı (Bounce Rate):** Öğrencilerin hangi ders içeriklerinde platformu terk ettiği.
* **Ders Tamamlama Süresi:** Bir öğrenme yolunun ortalama bitirilme süresi.
* **Aktif Kullanıcı Sayısı (DAU):** Günlük platform kullanımı.

## 2. A/B Test Tasarımı
İki farklı arayüz tasarımı karşılaştırılacaktır:
* **Versiyon A:** Liste şeklinde ders içeriği.
* **Versiyon B:** Kart (Grid) yapısında görselli ders içeriği.
* **Hedef:** Hangi arayüzde ders başlatma oranı daha yüksek?

## 3. Üçüncü Taraf API Entegrasyonu
Eğitim araçları için (Zoom, Kahoot vb.) API uç noktaları belirlenmiş olup, entegrasyon sonrası veri akışı doğrulanacaktır.
